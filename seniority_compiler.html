Step 2: Paste PAFCA Office Seniority Data

📥 How to get the data:

  1. Go to the PAFCA website and download the Excel version of the seniority list
  2. Open the Excel file and go to the "Office Seniority" tab
  3. ⚠️ IMPORTANT: Unhide the PPR column (Column D) - this contains Worker IDs needed for ASCENT export
  4. Select ALL data (Ctrl+A) including the header row
  5. Copy (Ctrl+C) and paste (Ctrl+V) into the box below
